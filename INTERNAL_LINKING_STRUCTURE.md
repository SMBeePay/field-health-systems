## üìä LINKING STRATEGY RULES

### DO's:
‚úÖ Link from blog posts UP to pillar pages (most important!)
‚úÖ Link from pillar pages DOWN to related blog posts
‚úÖ Link pillar pages to service/conversion pages
‚úÖ Use descriptive anchor text with target keywords
‚úÖ Link related blog posts to each other (lateral linking)
‚úÖ Every page should link to at least 3-5 other pages
‚úÖ Priority: Pillar pages get most internal links

### DON'Ts:
‚ùå Don't use "click here" or "read more" as anchor text
‚ùå Don't over-optimize anchor text (vary naturally)
‚ùå Don't link every mention of a keyword
‚ùå Don't create circular link chains
‚ùå Don't forget mobile navigation

## üîç TRACKING & OPTIMIZATION

### Monitor These Metrics:
- Click-through rate on internal links (GA4)
- Time on page (longer = better engagement)
- Pages per session (more = better linking)
- Bounce rate (lower = better internal links)
- Conversion rate by landing page
- Top exit pages (add better links there)

### Tools to Use:
- Google Analytics 4: Navigation Summary report
- Google Search Console: Internal link data
- Screaming Frog: Visualize link structure
- Ahrefs/SEMrush: Internal link opportunities
